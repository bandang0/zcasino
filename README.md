# zcasino
The `zcasino` game is a CLI roulette game. The command does not take any arguments and the player is guided through the game by the CLI.


### Installing and usage

To play `zcasino`:
* `git clone https://github.com/bandang0/zcasino`
* `cd zcasino`
* `./zcasino`
